# Course Booking API

##  Project Overview

Backend system built using Spring Boot for managing course offerings, sessions, and bookings with conflict detection.

---

##  Tech Stack

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Postman

---

##  How to Run

1. Clone repository
2. Import into IDE
3. Configure database in application.properties
4. Run Spring Boot application
5. Use Postman for API testing

---

##  Postman Collection

Postman collection JSON is included in this repository for testing APIs.

---

##  Features

* Create offerings
* Add sessions
* Book offerings
* Conflict detection
* Concurrent booking handling

---

##  API Testing

All APIs are available in Postman collection file inside repository.
